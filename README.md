## Warum das Gehalt immer knapper wird – obwohl sich nichts "Großes" verändert hat

Erkläre das mal jemandem, der keine Ahnung von Finanzen hat, und er schüttelt den Kopf: *Wie kann jemand monatlich Geld verdienen und trotzdem nie welches übrig haben?*

Ganz einfach – es liegt am Muster, nicht an der Zahl.

Das klassische Muster sieht so aus: Tag 1 bis 7, Geld kommt rein, alles ist entspannt. Woche zwei, der Alltag läuft, ein paar Ausgaben hier und da. Woche drei, man fängt an zu rechnen. Woche vier, das Konto geht auf Reserve. Und Woche eins nächsten Monats startet wieder von vorne.
